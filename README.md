# API + GUI + Ejecutable – Dragon Ball App

Esta aplicación consume la API pública de Dragon Ball y muestra información de los personajes.

##  Tecnologías utilizadas
- Python 3
- Requests (para consumir API)
- Pillow (para mostrar imágenes)
- Tkinter (para la GUI)
- PyInstaller (para generar el .exe)

##  API utilizada
Dragon Ball API:  
https://dragonball-api.com/api/characters

##  Funcionalidades
- Consume datos en formato JSON.
- Muestra nombre, raza, ki, género y foto del personaje.
- Interfaz desarrollada en Tkinter.
- Compilable como ejecutable con PyInstaller.
